========================================================================
    콘솔 응용 프로그램 : cppinvader 프로젝트 개요
========================================================================

응용 프로그램 마법사에서 이 cppinvader 응용 프로그램을 만들었습니다.

cppinvader 응용 프로그램을 구성하는 각 파일에 대한
요약 설명이 포함되어 있습니다.


cppinvader.vcxproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다. 파일을 생성한 Visual C++ 버전에 대한 정보와 응용 프로그램 마법사를 사용하여 선택한 플랫폼, 구성 및 프로젝트 기능에 대한 정보가 포함되어 있습니다.

cppinvader.vcxproj.filters
    응용 프로그램 마법사를 사용하여 생성된 VC++ 프로젝트의 필터 파일입니다. 이 파일에는 프로젝트의 파일과 필터 간의 연결 정보가 들어 있습니다. 이러한 연결은 특정 노드에서 유사한 확장명으로 그룹화된 파일을 표시하기 위해 IDE에서 사용됩니다. 예를 들어 ".cpp" 파일은 "소스 파일" 필터와 연결되어 있습니다.

cppinvader.cpp
    기본 응용 프로그램 소스 파일입니다.

C++객체 지향을 이용한 인베이더 게임 만들기

구조

1. 파서(파일입출력 : invader.txt)를 이용한 인베이더 난이도 조정

2. 자료구조 이중 리스트을 이용한 데이터 (몬스터, 미사일, 플레이어 등 )관리 

3. 클래스를 이용한 상속관계

4. 2단계까지 난이도 만듬


게임 사용법

F1 : 게임 시작

스페이스 : 공격 , 방향 : 이동
